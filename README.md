# DBRLearning

This repository aggregates resources, notebooks, and code for Databricks Runtime (DBR) learning courses and certification prep.

## Courses

### Data Engineering with Databricks
Path: `data-engineering-with-databricks/`

Complete course materials organized by module:
- **DE 1** - Databricks Workspace
- **DE 2** - ETL with Spark
- **DE 3** - Delta Lake
- **DE 4** - Delta Live Tables
- **DE 5** - Workflow Jobs
- **DE 6** - Managing Permissions

Each module includes:
- Lecture notebooks (`.py`)
- Hands-on lab notebooks
- Solutions directory with reference implementations
- Shared includes (setup scripts, dataset configs, pipeline utilities)

### Building Retrieval Agents on Databricks
Path: `retrieval-agents/`
Course: https://customer-academy.databricks.com/learn/courses/2706/building-retrieval-agents-on-databricks/

- `1.ai_parse.ipynb` - AI parsing and document processing notebook
- `document_renderer.py` - Document rendering utilities

### ML Associate Certification Prep
Path: `ML Associate/`

Practice notebooks for Databricks Certified Machine Learning Professional:
- `01 - Data Preparation: Feature Engineering and AutoML.py` - Feature engineering pipelines and AutoML setup
- `02 - Data Exploration: Generated by AutoML.py` - Automated EDA outputs
- `03 - Best Trial Run: XGBoost training improvements.py` - Model tuning and XGBoost optimization

## Usage

Notebooks are exported from Databricks workspace and can be:
- Imported back into Databricks for execution
- Reviewed locally for learning and reference
- Used as study materials for certification exams
